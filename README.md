## Pré-requisitos

- Java 17
- Maven
- Angular 16
- Node.js e npm compátiveis com a versão do Angular 

## Funcionalidades Principais
### Admin
    - CRUD de chamados
    - CRUD de assunto, categoria e comentário
    - CRUD de usuário
### Usuário
    - Cadastro e visualização dos chamados
